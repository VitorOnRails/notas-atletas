# Projeto de certificação DEVStart

## Técnologia utilizada:
![Javascript](./assets/js.png)
###  O projeto percorre, calcula e formata matrizes de notas de atletas. Faz o descarte da maior e menor nota e calcula a média das notas restantes, retornando a média válida da nota de cada atleta.

## Funcionalidades do projeto:
- [X] Recebe diferentes quantidades de notas de atletas.
- [X] Formata nomes e notas deixando mais legível.
- [X] Calcula a média válida, descartando a nota maior e menor.
- [X] Realiza tratamento para casos mais específicos.


## O código pode ser executado em:
Navegadores Web modernos: Chrome, Microsoft Edge, Safari, Brave, etc.
Ou Node.js.

## Como executá-lo?

### Execução em navegadores web:
- Salve o código em um arquivo com o nome notas-dos-atletas  e cole todo o código do projeto.
- Crie um arquivo .html
- Abra esse arquivo html em um navegador web moderno.
----------------------------------------------------------
### Execução no Node.js:
- Salve o código em um arquivo com o nome notas-dos-atletas.
- Execute no terminal.
